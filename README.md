<img src="https://user-images.githubusercontent.com/81938036/150527143-b483e367-0ff1-486e-8922-e6b554aefd03.png" width="256" height="509"> <img src="https://user-images.githubusercontent.com/81938036/150527145-47ba5f6c-ea45-439f-a29e-16f9ed5138a6.png" width="256" height="509"> <img src="https://user-images.githubusercontent.com/81938036/150527150-04707eb2-a5c6-4d9c-a4c8-33b7394e8ffd.png" width="256" height="509">

Pxloader is an android application for downloading images of Pixiv.  
Search illustrations and ugoiras by keyword, rankings, user id, work id.  
It also supports works of following users and collection.
<br><br>

## Features
### Download
<img src="https://user-images.githubusercontent.com/81938036/150549809-a68b8508-d685-4ffb-a14f-ac91b72d8adf.png" width="256" height="367">

- Search  
Search for and save the works with keywords.  
We can set options such as the number of likes and order.

- Rankings  
List and save the works with rankings.  
We can set options such as dates and categories.

- Following  
List and save the works of following users.  
Guests cannot use it.

- Collection  
List and save the bookmarked works.  
We can select an access spicifier(public/private).  
Guests cannot use it.

- User  
List and save the works of a specific user.  
We can enter user ID or get it from the Pixiv.

- Work ID  
Save the work of a specific work.  
We can enter work ID or get it from the Pixiv.
<br><br>

### Archive
<img src="https://user-images.githubusercontent.com/81938036/150527145-47ba5f6c-ea45-439f-a29e-16f9ed5138a6.png" width="256" height="509">
Display image files on the directory path.
<br><br>

### Settings
<img src="https://user-images.githubusercontent.com/81938036/150550364-ebd68a8a-e1fd-4431-a832-2e5d69afeec8.png" width="256" height="369">

- Create Subdirectory  
Create a subdirectory according to the download method or options.

- Directory Path  
Select a directory path to save image files.

- File Name Format  
Decide the file name format by listing various attributes of the work.

- Tags to Exclude  
Add tags to be excluded from the search results.
<br><br>

## Usage
### Search
<img src="https://user-images.githubusercontent.com/81938036/150551877-8acaca17-54e1-40fc-bbec-065e6be81728.png" width="256" height="398"> <img src="https://user-images.githubusercontent.com/81938036/150551885-27dcf25c-b25e-4784-be61-49b2b96fbb71.png" width="256" height="398">

1. Enter the search keyword(required).
2. Enter the minimum number of likes.
3. Select the sord order.
4. Select the age limit.
5. Click the download button.
<br><br>

### User
<img src="https://user-images.githubusercontent.com/81938036/150552651-5b746b6d-d462-443f-8913-6df870f71c37.png" width="256" height="398"> <img src="https://user-images.githubusercontent.com/81938036/150552659-c80fcb11-29e8-41a5-bc37-462050a9de89.png" width="256" height="398"> <img src="https://user-images.githubusercontent.com/81938036/150552661-4ca5656b-fa96-4f07-86b9-a56bf7b6e898.png" width="256" height="398">

1. Access the user page on Pixiv application.
2. Click the overflow button at the top right.
3. Select share.
4. Select the Pxloader.
5. Click the download button.
<br><br>

### Work ID
<img src="https://user-images.githubusercontent.com/81938036/150552995-45bf3973-881f-4ff5-8fa9-788bb298322c.png" width="256" height="398"> <img src="https://user-images.githubusercontent.com/81938036/150553002-818dec68-d386-4b6d-8416-58b0e3837f88.png" width="256" height="398">

1. Access the work page on Pixiv application.
2. Click the share button at the top right.
3. Select the Pxloader.
4. Click the download button.
